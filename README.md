# termux-aapt

## Prebuilt aapt and aapt2 for termux android

> This is a fork from  [iBotPeaches](https://github.com/iBotPeaches/platform_frameworks_base) and compile it to work on termux android

#### List of supported architectures:

- [x] arm64-v8a
- [x] armeabi-v7a
- [x] i686
- [x] x86_64
